3.2 管理员权限执行cmd窗口
运行1
npm config set prefix D:\WORK\NODEJS\node_config\node_global
默认情况下，npm 全局安装的模块路径是在 Node.js 的安装目录下，但通过设置这个路径，可以将全局模块安装在指定的目录下，方便管理和避免权限问题。
运行2
npm config set cache D:\WORK\NODEJS\node_config\node_cache
npm 在安装包时会将下载的文件缓存起来，以加快后续的安装速度。默认情况下，这些缓存文件存储在用户主目录下，通过设置这个路径，可以将缓存文件存储在指定的目录中，便于管理和维护。
